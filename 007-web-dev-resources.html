Web Dev Resources

A handful of resources I keep coming back to as I build websites. Click any one to open it in a new tab, so this page stays put while you read.

MDN Web Docs: the reference for HTML, CSS, and JavaScript.

Can I Use: check which browsers support a feature.

CSS-Tricks: deep articles and a handy CSS Almanac.

GitHub: host your code and collaborate with others.

CodePen: try out HTML, CSS, and JS in the browser.
