## Notes

### Best Practices

#### General

- `<!DOCTYPE html>` common convention is UPPERCASE
- `<title>` a11y and SEO
- 2 spaces for indentation
- Only one `<h1>` element per page
- Don't skip heading levels

#### Alt Attribute

- Always include it
- One short sentence description
- a11y
- Use `alt=""` for decorative images

#### Self-Closing Tags
- `/>` is optional
