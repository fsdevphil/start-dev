How I Set Up My Editor

A short tour of my code editor setup: the theme, the extensions, and the one keybinding I use more than any other.

The theme is high-contrast on purpose; I spend hours staring at this screen and anything washed-out is painful by the end of the day. The font is monospace, generously sized.

The one keybinding I'd never give up is "go to file": fuzzy-find any file in the project with a few keystrokes. Once you've used it, every other way of opening files feels slow.
