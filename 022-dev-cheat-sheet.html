Developer Cheat Sheet

A handful of commands and shortcuts I reach for constantly.

Useful commands

To check your Node version, run node --version. To list files in the current directory, run ls -la. Most CLI flags start with a dash, like --help or -v.

Start a new project

Create a folder, step into it, and initialize a new project:

mkdir my-project
cd my-project
npm init -y

Keyboard shortcuts

Save the current file: press Ctrl + S (or Cmd + S on Mac).

Copy: press Ctrl + C.

Paste: press Ctrl + V.
